# 有机建筑

> 有机建筑（Organic Architecture）是一种强调建筑与自然环境、使用功能和材料本质相融合的设计哲学。其核心理念是建筑应像生命体一样从内而外地"生长"，而不是强加于场地。

---

## 核心理念

| 原则 | 说明 |
|------|------|
| **建筑属于场地** | 建筑应源于场地特征，与景观融为一体 |
| **从内而外** | 空间决定形式，而非形式决定空间 |
| **材料的诚实** | 展现材料的自然特性（砖的纹理、木的温暖、混凝土的质感） |
| **连续性** | 内部空间与外部环境的无缝过渡 |
| **整体性** | 建筑、家具、景观应作为一个整体设计 |

## 代表案例

- **[[流水别墅]]**（赖特）— 悬挑于瀑布之上，建筑与岩石、水流融为一体
- **西塔里埃森**（赖特）— 用当地石材和木材建造，宛如从沙漠中生长出来
- **MIT Baker House**（阿尔托）— 波浪形立面回应河景
- **帕米奥疗养院**（阿尔托）— 建筑如手指般伸入森林

---

## 与其他概念的关系

有机建筑并不反对 [[现代主义建筑]]，而是对其机械功能主义的一种修正和补充。它强调建筑不应只是"居住的机器"，还应是人与自然的媒介。
