# 截角正方形镶嵌 4.8.8（地中海式）

![[wiki/tiling/attachments/truncated_square_tiling.svg]]

## 基本原理

**4.8.8**（截角正方形镶嵌）是 8 种阿基米德镶嵌之一。每个顶点由**一个正方形 + 两个正八边形**构成，内角和 = 90° + 135° + 135° = 360°。

- **Schläfli 符号**：t{4,4}（对正方形镶嵌进行截角操作）
- **对称群**：p4m，[4,4]，(*442)
- **对偶镶嵌**：四角正方形镶嵌（Tetrakis square tiling）
- **别名**：地中海式镶嵌（Mediterranean tiling）、八边形镶嵌

## 构造方法

对正方形镶嵌进行**截角（Truncation）**——将正方形的每个角切掉，正方形变成八边形，原顶点处产生新的小正方形。

这也是唯一含正八边形的边对边正则多边形镶嵌。

## 适合镂空板的原因

| 特点 | 说明 |
|:---|:---|
| ✅ **现代感强** | 八角形 + 正方形的组合简洁大气 |
| ✅ **对称性好** | 四边形对称，容易与建筑模数对齐 |
| ✅ **加工友好** | 所有角度 ≥ 90°，无尖角，强度好 |
| ✅ **应用广泛** | 常见于地中海风格地砖和建筑装饰 |
| ⭐ **推荐** | **现代风格镂空板的首选，简洁且容易加工** |

## 镂空板设计提示

1. 八边形做**镂空孔洞**，正方形保留为**连接结构**
2. 反之亦可——正方形做孔，八边形保留
3. 适合配合 CNC / 激光切割，不存在加工死角

## Grasshopper 实现

```
LunchBox → Patterns → Grid Based → Square Grid
    → Offset 每个单元 → 识别八边形和正方形
    ─ 或 ─
手动构建：正方形阵列 → 对每个角做45°切角
```
