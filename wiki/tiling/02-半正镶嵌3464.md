# 半正镶嵌 3.4.6.4（小斜方截半）

![[wiki/tiling/attachments/rhombitrihexagonal_tiling.svg]]

## 基本原理

**3.4.6.4** 是 8 种阿基米德（半正）镶嵌之一。围绕每个顶点依次是：**三角形 → 正方形 → 六边形 → 正方形**，内角和 = 60° + 90° + 120° + 90° = 360°。

- **Schläfli 符号**：rr{6,3} 或 rr{3,6}（对六边形镶嵌进行"截角+扩展"操作）
- **对称群**：p6m，(*632) — 与六边形镶嵌相同
- **对偶镶嵌**：筝形三角六边形镶嵌（Deltoidal trihexagonal tiling）
- **中文名**：小斜方截半六边形镶嵌

## 构造方法

由正六边形镶嵌通过**康特雷（Cantellation）**操作获得：将每个六边形的边向外扩展，六边形变成正六边形，顶点处产生正三角形，边处产生正方形。

## 适合镂空板的原因

| 特点 | 说明 |
|:---|:---|
| ✅ **三种图形混合** | 三角形 + 正方形 + 六边形，节奏丰富 |
| ✅ **伊斯兰感** | 常出现在伊斯兰几何装饰中，有传统美感 |
| ✅ **结构连通** | 各多边形交错连接，镂空后不会散架 |
| ⭐ **推荐** | **平衡了丰富性与加工可行性，是最适合装饰性镂空的半正镶嵌** |
