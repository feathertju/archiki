# 正六边形镶嵌（蜂巢）

![[wiki/tiling/attachments/Hexagonal_tiling.svg]]

## 基本原理

正六边形镶嵌是三种**正镶嵌**（Regular Tessellation）之一，也是最著名的一种。每个顶点由 3 个正六边形相遇，内角和 = 120° × 3 = 360°。

- **Schläfli 符号**：{6,3}（6 表示正六边形，3 表示每个顶点有 3 个图形相遇）
- **对称群**：p6m，[6,3]，(*632)
- **对偶镶嵌**：正三角形镶嵌 {3,6}

## 蜂巢猜想（Honeycomb Conjecture）

在同等面积下，正六边形是用**最短总周长**分割平面的方式——这就是为什么蜜蜂用六边形筑巢，自然界中石墨烯、碳纳米管也是这种结构。

## 适合镂空板的原因

| 特点 | 说明 |
|:---|:---|
| ✅ **结构强度最高** | 六边形框架受力均匀，不易变形 |
| ✅ **无尖角** | 120° 内角，加工不易断裂 |
| ✅ **重复单元简单** | 只需一个六边形即可无限阵列 |
| ✅ **视觉经典** | 蜂巢图案有自然感和科技感 |
